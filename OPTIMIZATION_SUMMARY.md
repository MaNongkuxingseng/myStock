## 🚀 核心功能详解

### 1. 实时信号分析系统

#### 评分规则（0-100分）
```
涨跌幅 (±20分)
  - 涨幅>3%: +20分
  - 涨幅>1%: +10分
  - 跌幅<-3%: -20分
  - 跌幅<-1%: -10分

量比 (+15分)
  - 量比>2: +15分
  - 量比>1.5: +10分

资金流向 (+15分)
  - 当日净流入: +10分
  - 3日净流入: +5分

累计涨幅 (+10分)
  - 3日涨幅>5%: +10分
  - 3日涨幅>2%: +5分

估值惩罚 (-8分)
  - PE>50: -5分
  - PB>5: -3分
```

#### 信号级别
- **70+分**: 🟢 强烈买入
- **60-69分**: 🟡 买入  
- **50-59分**: ⚪ 观望
- **40-49分**: 🟠 谨慎
- **<40分**: 🔴 卖出

### 2. 操作建议系统

#### 买入建议
```python
# 强烈买入 (评分≥70)
买入价位: 当前价×0.99
止损价位: 当前价×0.93
目标价位: 当前价×1.15

# 买入 (评分60-69)
买入价位: 当前价×0.97
止损价位: 当前价×0.92
目标价位: 当前价×1.12
```

#### 其他建议
- **观望**: 等待明确信号
- **谨慎**: 控制仓位，轻仓试探
- **卖出**: 考虑减仓或离场

## 🗄️ 数据库优化

### 新增表结构
```sql
CREATE TABLE stock_realtime (
    id INT AUTO_INCREMENT PRIMARY KEY,
    stock_code VARCHAR(10) NOT NULL,
    stock_name VARCHAR(50),
    current_price DECIMAL(10,2),
    change_rate DECIMAL(10,2),
    volume BIGINT,
    amount DECIMAL(15,2),
    update_time DATETIME NOT NULL,
    data_source VARCHAR(20),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_stock_code (stock_code),
    INDEX idx_update_time (update_time)
);
```

### 数据流
```
实时数据源 → 数据获取 → 信号分析 → 报告生成 → 操作建议
    ↓
数据库存储 ← 数据验证 ← 结果缓存
```
